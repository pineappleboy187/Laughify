Example API Request:

fetch('https://api.laughify.com/jokes/random')

API Documentation

  • GET /jokes/random: Returns a random joke.
  • GET /jokes/{category}: Returns a random joke from the specified category.
  • GET /categories: Lists all available joke categories.

Sample Response:

{
    "category": "dad joke",
    "joke": "Why did the scarecrow win an award? Because he was outstanding in his field!"
}
